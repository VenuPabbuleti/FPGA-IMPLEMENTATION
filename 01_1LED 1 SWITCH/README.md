# LED BLINKING
  This includes Interfacing of LED using Switch i.e Nothing but a Buffer
  
    ==> Switch - HIGH(1) ---> LED Turn on
    
    ==> Switch - LOW(0)  ---> LED Turn off

##  In FPGA Board
    Switch(Input) --> SWITCH[0]
    
    LED (Output)  --> LED[0]
